# Troubleshooting & FAQ

Issues and solutions for common ProphetBase problems.

## Common Issues

### 1. Wallet Connection Fails
**Symptoms**: "Connect Wallet" button does nothing or returns an error.
**Solutions**:
- Ensure you have a Stacks-compatible wallet extension installed (e.g., Hiro, Leather).
- Refresh the page and try again.
- Clear browser cache for this site.

### 2. Transaction "Pending" for a long time
**Symptoms**: Transaction submitted but not appearing as confirmed.
**Reason**: Network congestion or low gas fee.
**Solutions**:
- Wait up to 10-15 minutes (Stacks block times vary).
- Use a "Speed up" function in your wallet if available.
- Check Stacks Explorer for real-time status.

### 3. Incorrect Balance Display
**Symptoms**: Wallet shows different balance than the dashboard.
**Solutions**:
- Ensure you are on the correct network (Mainnet vs Testnet).
- Refresh dashboard using the "Sync" button.
