# Connecting Your Wallet

## Supported Wallets
- **MetaMask** - Browser extension and mobile app
- **Coinbase Wallet** - Native Base support
- **Rainbow** - Mobile-first wallet
- **WalletConnect** - Any WalletConnect-compatible wallet

## Connection Steps

### Desktop (Browser Extension)
1. Click **Connect Wallet** button
2. Select your wallet from the modal
3. Approve the connection request in your wallet
4. Confirm you're on Base network

### Mobile
1. Open ProphetBase in your wallet's browser
2. Tap **Connect Wallet**
3. Approve the connection
4. Ensure you're on Base Mainnet

## Network Configuration

### Base Mainnet
- **Network Name**: Base
- **RPC URL**: https://mainnet.base.org
- **Chain ID**: 8453
- **Currency**: ETH
- **Block Explorer**: https://basescan.org

## Troubleshooting

### Wallet Not Connecting
- Refresh the page
- Clear browser cache
- Try a different wallet
- Check if wallet is unlocked

### Wrong Network
- Click the network switcher in your wallet
- Select "Base" or add it manually using the config above
