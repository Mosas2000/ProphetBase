# Getting Started with ProphetBase

## Overview
ProphetBase is a decentralized prediction market platform built on Base. Users can create markets, buy shares, and earn rewards based on accurate predictions.

## Prerequisites
- A Web3 wallet (MetaMask, Coinbase Wallet, or Rainbow)
- ETH on Base network for gas fees
- USDC for trading (optional)

## Quick Start

### 1. Visit the Platform
Navigate to [prophetbase.app](https://prophetbase.app)

### 2. Connect Your Wallet
Click "Connect Wallet" in the top right corner and select your preferred wallet.

### 3. Switch to Base Network
If you're not on Base, your wallet will prompt you to switch networks.

### 4. Start Trading
Browse active markets and start making predictions!
